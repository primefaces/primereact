## Usage

```tsx
import { IconField } from 'primereact/iconfield';
```

```tsx
<IconField>
    <IconField.Icon />
    <InputText />
</IconField>
```

## Accessibility

### Screen Reader

IconField and InputIcon do not require any roles and attributes.

### Keyboard Support

Components does not include any interactive elements.
