## ProgressSpinner

| Name | Type | Default | Description |
|----------|------|---------|-------------|
| _strokeWidth_ | `string \| undefined` | 2 | Width of the circle stroke. |
| _fill_ | `string \| undefined` | undefined | Color for the background of the circle. |
| _animationDuration_ | `string \| undefined` | 2s | Duration of the rotate animation. |
