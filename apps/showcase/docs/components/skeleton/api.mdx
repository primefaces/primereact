## Skeleton

| Name | Type | Default | Description |
|----------|------|---------|-------------|
| _shape_ | `'rectangle' \| 'circle' \| undefined` | rectangle | Shape of the element. |
| _size_ | `string \| undefined` | undefined | Size of the Circle or Square. |
| _width_ | `string \| undefined` | 100% | Width of the element. |
| _height_ | `string \| undefined` | 1rem | Height of the element. |
| _borderRadius_ | `string \| undefined` | undefined | Border radius of the element, defaults to value from theme. |
| _animation_ | `'wave' \| 'none' \| undefined` | wave | Type of the animation. |
