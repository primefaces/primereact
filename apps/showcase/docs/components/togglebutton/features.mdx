---
title: 'ToggleButton'
description: 'ToggleButton component is used to create a button that can be toggled on or off.'
component: 'togglebutton'
---

## Usage

```tsx
import { ToggleButton } from 'primereact/togglebutton';
```

```tsx
<ToggleButton>
    <ToggleButton.Indicator>Toggle</ToggleButton.Indicator>
</ToggleButton>
```
