## Accessibility

### Screen Reader

Knob element component uses slider role in addition to the _aria-valuemin_, _aria-valuemax_ and _aria-valuenow_ attributes. Value to describe the component can be defined using _aria-labelledby_ and _aria-label_ props.

### Keyboard Support

| Key               | Function                          |
| ----------------- | --------------------------------- |
| `tab`             | Moves focus to the knob.          |
| `left/down arrow` | Decrements the value.             |
| `right/up arrow`  | Increments the value.             |
| `home`            | Set the minimum value.            |
| `end`             | Set the maximum value.            |
| `page up`         | Increments the value by 10 steps. |
| `page down`       | Decrements the value by 10 steps. |
