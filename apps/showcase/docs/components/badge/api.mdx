## Badge

| Name | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| _severity_ | `'secondary' \| 'info' \| 'success' \| 'warn' \| 'danger' \| 'contrast' \| undefined` | undefined | Severity type of the badge. |
| _size_ | `'small' \| 'large' \| 'xlarge' \| undefined` | undefined | Size of the badge, valid options are 'small', 'large', and 'xlarge'. |

## Headless

### useBadgeProps

| Name | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| _value_ | `string \| number \| undefined` | undefined | Value of the badge. |
