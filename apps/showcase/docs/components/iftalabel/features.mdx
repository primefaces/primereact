## Usage

```tsx
import { Label } from 'primereact/label';
```

```tsx
<Label.Ifta>
    <InputText />
    <Label></Label>
</Label.Ifta>
```

### Screen Reader

IftaLabel does not require any roles and attributes.

### Keyboard Support

Component does not include any interactive elements.
