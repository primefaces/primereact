## Usage

```tsx
import { Label } from 'primevue/label';
```

```tsx
<Label.Ifta>
    <InputText />
    <Label></Label>
</Label.Ifta>
```

## Accessibility

### Screen Reader

IftaLabel does not require any roles and attributes.

### Keyboard Support

Component does not include any interactive elements.
