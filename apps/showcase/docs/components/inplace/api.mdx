## Headless

### useInplaceChangeEvent

| Name | Type | Default | Description |
|----------|------|---------|-------------|
| _active_ | `boolean` | undefined | The active state of the inplace. |

### useInplaceProps

| Name | Type | Default | Description |
|----------|------|---------|-------------|
| _active_ | `boolean \| undefined` | undefined | Whether the content is displayed or not. |
| _disabled_ | `boolean \| undefined` | undefined | When present, it specifies that the element should be disabled. |
| _onActiveChange_ | `(active: boolean \| undefined) => void` | undefined | Callback function that is called when the element is clicked. |
