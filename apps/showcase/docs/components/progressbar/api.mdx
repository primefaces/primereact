## ProgressBar

| Name | Type | Default | Description |
|----------|------|---------|-------------|
| _mode_ | `'determinate' \| 'indeterminate' \| undefined` | determinate | Defines the mode of the progress |
| _style_ | `React.CSSProperties \| undefined` | undefined | Style of the progressbar. |


## Headless

### useProgressBarProps

| Name | Type | Default | Description |
|----------|------|---------|-------------|
| _value_ | `number \| undefined` | undefined | Current value of the progress. |
