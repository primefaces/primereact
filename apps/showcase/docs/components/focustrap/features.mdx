---
title: Focus Trap
description: Focus Trap keeps focus within a certain DOM element while tabbing.
component: focustrap
---

## Usage

```tsx
import { FocusTrap } from 'primereact/focustrap';
```

```tsx
<FocusTrap container={}>
    <div>
        <input type="text" />
    </div>
</FocusTrap>
```
