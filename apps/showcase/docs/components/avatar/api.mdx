## AvatarFallback

| Name | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| _label_ | `string \| undefined` | undefined | The label of the avatar fallback. |
| _icon_ | `string \| undefined` | undefined | The icon of the avatar fallback. |

## Avatar

| Name | Type | Default | Description |
|:-----|:-----|:--------|:------------|
| _size_ | `string \| undefined` | normal | Defines the size of the avatar. |
| _shape_ | `string \| undefined` | square | Defines the shape of the avatar. |
| _ariaLabel_ | `string \| undefined` | undefined | Establishes a string value that labels the component. |
| _ariaLabelledby_ | `string \| undefined` | undefined | Establishes relationships between the component and label(s) where its value should be one or more element IDs. |
