## TagIcon

| Name | Type | Default | Description |
|----------|------|---------|-------------|
| _icon_ | `string \| undefined` | undefined | The icon to display in the tag. |

## Tag

| Name | Type | Default | Description |
|----------|------|---------|-------------|
| _severity_ | `'secondary' \| 'success' \| 'info' \| 'warn' \| 'danger' \| 'contrast' \| undefined` | undefined | Severity type of the tag. |
| _rounded_ | `boolean \| undefined` | false | Whether the corners of the tag are rounded. |
