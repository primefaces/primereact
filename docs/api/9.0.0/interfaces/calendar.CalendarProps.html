Optional appendTo

appendTo?: null | HTMLElement | "self"

DOM element instance where the overlay panel should be mounted. Valid values are any DOM Element and "self". The "self" value is used to render a component where it is located.

Default Value

document.body

Optional ariaLabelledBy

ariaLabelledBy?: string

Establishes relationships between the component and label(s) where its value should be one or more element IDs.

Optional autoZIndex

autoZIndex?: boolean

Whether to automatically manage layering.

Default Value

true

Default Value

0

false

Whether the month should be rendered as a dropdown instead of text.

since v6. Navigator is always on.

Default Value

false

Inline style of the datetimepicker panel.

Optional placeholder

placeholder?: string

Placeholder text for the input.

Optional readOnlyInput

readOnlyInput?: boolean

When specified, prevents entering the date manually with keyboard.

Default Value

false

Optional required

required?: boolean

When present, it specifies that an input field must be filled out before submitting the form.

Default Value

false

Optional selectOtherMonths

selectOtherMonths?: boolean

Whether days in other months shown before or after the current month are selectable. This only applies if the showOtherMonths option is set to true.

Default Value

false

Optional selectionMode

selectionMode?: "multiple" | "range" | "single"

Specifies the selection mode.

Default Value

single

Optional shortYearCutoff

shortYearCutoff?: string

The cutoff year for determining the century for a date.

Default Value

+10

Optional showButtonBar

showButtonBar?: boolean

Whether to display today and clear buttons at the footer

Default Value

false

Optional showIcon

showIcon?: boolean

When enabled, displays a button with icon next to input.

Default Value

false

Optional showMillisec

showMillisec?: boolean

Whether to show the milliseconds in time picker.

Default Value

false

Optional showMinMaxRange

showMinMaxRange?: boolean

Whether to allow navigation past min/max dates.

Default Value

false

Optional showOnFocus

showOnFocus?: boolean

When disabled, datepicker will not be visible with input focus.

Default Value

true

Optional showOtherMonths

showOtherMonths?: boolean

Whether to display dates in other months (non-selectable) at the start or end of the current month. To make these days selectable use the selectOtherMonths option.

Default Value

true

Optional showSeconds

showSeconds?: boolean

Whether to show the seconds in time picker.

Default Value

false

Optional showTime

showTime?: boolean

Whether to display timepicker.

Default Value

false

Optional showWeek

showWeek?: boolean

When enabled, calendar will show week numbers.

Default Value

false

Milliseconds to change per step.

Whether to display timepicker only.

Style class of the today button.

tooltip?: string

Content of the tooltip.

Configuration of the tooltip, refer to the tooltip documentation for more information.

When enabled, calendar overlay is displayed as optimized for touch devices.

Default Value

false

Optional transitionOptions

transitionOptions?: CSSTransitionProps<undefined>

Value of the component.

null

Optional view

view?: "month" | "date"

Type of view to display.

Default Value

date

Optional viewDate

viewDate?: null | Date

Date instance whose month and year are used to display the calendar.

Optional visible

visible?: boolean

Specifies the visibility of the overlay.

Default Value

false

Optional yearNavigator

yearNavigator?: boolean

Whether the year should be rendered as a dropdown instead of text.

Deprecated

since v6. Navigator is always on.

Default Value

false

Optional yearRange

yearRange?: string

The range of years displayed in the year drop-down in (nnnn:nnnn) format such as (2000:2020).
