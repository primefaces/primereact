Optional contextMenuSelectionKey

contextMenuSelectionKey?: string

A single key to control the selection with the context menu.

Optional disabled

disabled?: boolean

When present, it specifies that the component should be disabled.

Default Value

false

Optional dragdropScope

dragdropScope?: string

Unique key to enable dragdrop functionality.

Default Value

false

Optional expandedKeys

expandedKeys?: TreeExpandedKeysType

An array of keys to represent the state of the tree expansion state in controlled mode.

Optional filter

filter?: boolean

When specified, displays an input field to filter the items.

Default Value

false

Optional filterBy

filterBy?: string

When filtering is enabled, filterBy decides which field or fields (comma separated) to search against.

Default Value

label

Optional filterLocale

filterLocale?: string

Locale to use in filtering. The default locale is the host environment's current locale.

Default Value

undefined

Optional filterMode

filterMode?: "strict" | "lenient"

Mode for filtering valid values are "lenient" and "strict". Default is lenient.

Default Value

lenient

Optional filterPlaceholder

filterPlaceholder?: string

Placeholder text to show when filter input is empty.

Optional filterTemplate

filterTemplate?: ReactNode | ((options: TreeFilterOptions) => ReactNode)

Template of filter element.

Optional filterValue

filterValue?: string

When filtering is enabled, the value of input field.

Optional footer

footer?: ReactNode | ((props: TreeProps) => ReactNode)

The template of header.

Optional header

header?: ReactNode | ((options: TreeHeaderTemplateOptions) => ReactNode)

The template of header.

Optional id

id?: string

Unique identifier of the element.

Optional loading

loading?: boolean

Whether to display loading indicator.

Default Value

false

Optional loadingIcon

loadingIcon?: string

Icon to display when tree is loading.

Default Value

pi pi-spin

Optional metaKeySelection

metaKeySelection?: boolean

Defines how multiple items can be selected, when true metaKey needs to be pressed to select or unselect an item and when set to false selection of each item can be toggled individually. On touch enabled devices, metaKeySelection is turned off automatically.

Default Value

true

Optional nodeTemplate

nodeTemplate?: ReactNode | ((node: default, options: TreeNodeTemplateOptions) => ReactNode)

Template of node element.

Default Value

false

Optional propagateSelectionDown

propagateSelectionDown?: boolean

Whether checkbox selections propagate to descendant nodes.

Default Value

true

Optional propagateSelectionUp

propagateSelectionUp?: boolean

Whether checkbox selections propagate to ancestor nodes.

Default Value

true

Optional selectionKeys

selectionKeys?: null | string | TreeMultipleSelectionKeys | TreeCheckboxSelectionKeys

A single or an array of keys to control the selection state.

Optional selectionMode

selectionMode?: "multiple" | "checkbox" | "single"

Defines the selection mode, valid values "single", "multiple", and "checkbox".

Optional showHeader

showHeader?: boolean

Whether to show the header or not.

Default Value

true

Optional style

style?: CSSProperties

Inline style of the component.

Optional togglerTemplate

togglerTemplate?: ReactNode | ((node: default, options: TreeTogglerTemplateOptions) => ReactNode)

Template of toggler element.

Optional value

value?: default[]

An array of treenodes.
